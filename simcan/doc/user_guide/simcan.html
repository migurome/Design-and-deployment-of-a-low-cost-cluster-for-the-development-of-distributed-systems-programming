HOW TO BUILD AN APPLICATION

This section provides the steps neccesaries build an application in SIMCAN.

Build an application

  • SIMCAN provides a repository of applications
  • Each one of these applications contain a set of parameters that define the behavior of the application.
  • The same application can be modeled by using different configurations, that is, using different values to set its parameters.
  • In SIMCAN, each configuration of a given application is called instance.
  • Thus, in order to provide flexibility for modelling different environments, the SIMCAN repository is able to allocate different instances of several applications.
